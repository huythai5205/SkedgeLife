# SkedgeLife

Let individuals can easily create and schedule classes or sign up for classes such as yoga, coding, massages...etc.


###Demo
https://calm-waters-72414.herokuapp.com/

##Technologies Used:

###Front-end:
* ReactJs
* Redux
* materializeCSS

###Back-end:
* NodeJs
* ExpressJs
* mongoDB 
* mongoose


###Installation:

run 'npm install'

cd client

npm install

cd ..

run 'npm run start'
